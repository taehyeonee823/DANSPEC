# DANSPEC
## 서비스 소개
단국대에 재학생들을 대상으로 공모전, 대외 활동 등 다양한 활동을 맞춤형으로 추천하고, 이를 바탕으로 팀을 모집하고 지원할 수 있는 팀 매칭 서비스
## 📅 프로젝트 기간 및 팀원
### 프로젝트 기간
2025.11 ~ 2025.12
#### 프로젝트 팀원
|name|position|
|------|---|
|김태현|프론트엔드|
|신동은|프론트엔드|

### 🎯주요 기능
1. 맞춤형 활동 추천 

2. 팀 매칭 기능

3. 활동 탐색 & 챗봇 지원

4. 사용자 활동 분석

## 설치 및 실행
1. 레포지토리 클론
```bash
git clone https://github.com/taehyeonee823/DANSPEC.git
cd rn-starter
```
2. 설치 및 실행
```bash
npm install
npm run start
```
3. Expo Go 앱으로 QR 코드 스캔 후 실행
